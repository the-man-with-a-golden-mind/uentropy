Welcome

This is a single-page application built with entropy and a tiny hash-based router — no external routing library needed.

The router lives entirely in ~30 lines of JavaScript. It listens to hashchange, parses the current hash, stores the matched route and its params as reactive data, and entropy takes care of showing and hiding pages.
